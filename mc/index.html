На этой странице собраны если не все, то большая часть моих проектов для Minecraft.
LunarCore
Сервер под началом MaxLunar, который общими силами приводится к нашему пониманию идеала геймплея формата Vanilla+. На сервер можно попасть без каких-либо модов, но всё же рекомендуется собрать хотя-бы минимальную сборку для игры.

Карта сервера также находится в активной стадии разработки и строительства.

[Карта сервера]
project3x
Ресурс-пак, целью которого является сделать текстуры интерфейса pixel pefrect для игры с GUI Scale = 3.
Заменяются окна интерфейсов, элементы UI по типу здоровья, голода, опыта, и т.д.

На данный момент ресурс-пак очень неохотно допиливается и готовится к первому релизу.
pixelqiwi additions
Ресурс-пак, добавляющий новые скины для мобов, оружий, предметов, и пр.

На данный момент был заброшен при попытке портирования на последнюю версию с помощью дата-пака.
