Это мой проект для Minecraft версии 1.14.4 (и возможно выше), который добавляет различные косметические штуки в игру, и делает существующие штуки чуть лучше.

На данный момент этот пак заменяет почти весь интерфейс, в том числе заменяет дефолтный пиксельный шрифт на сексуальный truetype шрифт используя ванильные механики игры. Также делает некоторые вещи, которые занимали по пол экрана меньше, например щиты. Также этот пак добавляет нормальное такое количество скинов для предметов и мобов в игру.
